## Setup

```js
yarn install.
```

Install the adonis

```js
yarn add @adonisjs/cli
```

### Docker

```js
docker-compose up db
```

### Migrations

Run the following command to run startup migrations.

```js
yarn adonis migration:run
```

### Run 

```js
yarn adonis server --dev
```

### Test

```js
yarn adonis test
```

### Documentation

Na pasta /docs encontra-se documentação das apis do projeto.
